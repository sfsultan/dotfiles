### Installing docker in any other than default directory
```bash
start /w "" "Docker Desktop Installer.exe" install -accept-license  --installation-dir=E:\path\to\folder
```
```bash
start /w "" "Docker Desktop Installer.exe" install -accept-license  --installation-dir=E:\path\to\folder --wsl-default-data-root=E:\path\to\data\folder
```
